## Overview

To-Pra is a comprehensive productivity application that integrates task management, focus techniques, and gamification to enhance efficiency. Achieve your daily goals while earning rewards!

## Key Features

### 📋 Task Management
- Interactive to-do list with prioritization
- Quick task creation and editing
- Dynamic ✔️ completion marks
- Organization by categories or projects

### ⏱️ Focus Mode
- Customizable Pomodoro timer (25/50/90 mins)
- Visual and sound alerts when sessions end
- Automatic tracking of focused work periods
- Optional distraction blocker

### 📊 Detailed Analytics
- **Daily Dashboard**: Hourly activity charts and completion tracking
- **Full History**: Weekly/monthly productivity trends
- **Metrics**: Effective vs. estimated time analysis
- **Data Export**: CSV/PDF formats

### 🏆 Gamification System
- Earn points for completing tasks ✅
- Bonus rewards for productivity streaks 🔥
- Unlockable achievements (Novice, Expert, Master) 🏅
- Coming soon: Competitive weekly ranking 👑

## Technologies Used

- **Next.js**: For fast, server-rendered React applications.
- **Figma**: For creating and managing design assets and prototypes.
- **Tailwind CSS**: For styling React components with a modular, themable approach.


## Challenges and Learnings

One of the main challenges was balancing feature complexity with usability. Ensuring a seamless experience while integrating task management, focus modes, and gamification required an intuitive UI and efficient performance optimization.

Additionally, designing an engaging reward system without causing distractions involved careful iteration and user feedback. Implementing customizable Pomodoro timers and distraction-blocking mechanisms helped refine the app's effectiveness in boosting productivity.

## Outcome

| Metric | Result |
|--------|--------|
| **Active users** | 500+ monthly |
| **Tasks completed** | 12,000+ tracked |
| **Avg. focus sessions/user** | 4.2 per day |

To-Pra has successfully improved users' productivity by providing structured task management, time-tracking insights, and motivation through gamification. The application continues to evolve with competitive rankings and enhanced analytics.
