# Once UI Patterns

## Componentes disponibles
Los componentes de Once UI están en `src/once-ui/`

## Uso común
```tsx
import { Flex, Text, Button, Avatar } from '@/once-ui/components';

<Flex direction="column" gap="m">
  <Text variant="heading-strong-l">Título</Text>
  <Text variant="body-default-m">Contenido</Text>
  <Button variant="primary">Acción</Button>
</Flex>
```

## Tokens de diseño
- Spacing: xs, s, m, l, xl
- Variants de texto: heading-strong-l, body-default-m, etc.
