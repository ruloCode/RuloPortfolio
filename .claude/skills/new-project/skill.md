# Agregar Nuevo Proyecto al Portfolio

## Ubicación
`src/app/work/projects/`

## Crear archivo MDX
```mdx
---
title: "Nombre del Proyecto"
publishedAt: "2024-01-01"
summary: "Descripción corta del proyecto"
images:
  - "/images/projects/nombre/cover.jpg"
team:
  - name: "Rulo"
    role: "Developer"
    avatar: "/images/avatar.jpg"
---

## Descripción

Contenido del proyecto...

## Tecnologías

- Next.js
- React
- etc.
```

## Agregar imágenes
1. Crear carpeta en `public/images/projects/nombre-proyecto/`
2. Agregar cover.jpg y screenshots
